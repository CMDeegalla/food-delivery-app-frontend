FoodGo aims to be usable by everyone, including people with disabilities.

## Goals
- Provide a clear and easy-to-use interface
- Support users with low vision, motor difficulties, and reading challenges
- Follow accessibility good practices in layout, contrast, and interaction

## Implemented Practices
- Simple navigation and consistent page layout
- Readable text and clear buttons
- Sufficient contrast between text and background
- Forms and buttons designed with meaningful labels
- Reduced visual clutter to improve focus

## Feedback
If you find an accessibility issue, please raise it as a GitHub Issue in this repository.
